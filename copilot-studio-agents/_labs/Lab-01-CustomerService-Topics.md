# Lab 01 — Customer Service Assistant Topics (30–40 minutes)

## Goal
Implement three topics: returns/refunds, order status, and escalation.

## Files you will use
- `customer-service-assistant/topics/T01_ReturnsAndRefunds/`
- `customer-service-assistant/topics/T02_OrderStatus/`
- `customer-service-assistant/topics/T03_EscalateToHuman/`

## Outcome
Learners can:
- capture an order number
- answer policy questions using knowledge files
- generate a crisp escalation summary for a human
