# Contoso Electronics Returns Policy (Sample Knowledge)

**Effective date:** 2025-01-01  
**Applies to:** Contoso Electronics online orders (US)

## Quick rules
- **30-day return window** from delivery date for most items.
- **Refund method:** original payment method after inspection.
- **Restocking fee:** none for unopened items; up to **15%** for opened items where permitted by law.
- **Shipping:** Contoso pays return shipping for **defective** items; customer pays for **change-of-mind** returns.

## Exclusions
- Final-sale items.
- Downloadable software keys after redemption.
- Gift cards.

## How to return
1. Locate the order number in your confirmation email.
2. Start a return request with **Order Number + Billing ZIP**.
3. Print the prepaid label (if eligible) and pack the item with all accessories.
4. Drop off with the carrier and keep the receipt.

## Warranty vs return
A return is about satisfaction in the first 30 days.  
A warranty claim is about defects after the return window.
