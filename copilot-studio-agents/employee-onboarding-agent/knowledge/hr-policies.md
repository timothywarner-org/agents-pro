# Contoso HR Policies (Sample Knowledge)

## Acceptable Use (AUP)
Use company devices and accounts primarily for business needs.  
Do not install unapproved software.  
Report suspected security incidents immediately.

## Code of Conduct
Treat others with respect.  
Harassment and discrimination are not tolerated.  
Escalate concerns to HR or your manager.

## Remote Work
Remote work requires manager approval.  
Use VPN when accessing internal resources from offsite networks.
