# Course Plan

## Segment 1 - Using AI Agents (60 Minutes)
- Kickoff: align success metrics with promises from marketing and proposal collateral.
- Rapid walkthrough of Claude Code, GitHub coding agent, Microsoft Copilot chat, and Office add-ins to frame capability spectrum.
- Live comparison: map agent strengths to customer service, HR, and document automation scenarios.
- Environment readiness check: confirm Copilot Studio access, sample data, and Teams deployment targets.
- Prep work: assign learners to solution tracks and clarify expectations for the three builds.
- Q&A (5 minutes) and break (10 minutes).

## Segment 2 - Customer Service Assistant Build (60 Minutes)
- Define service goals and required knowledge assets with participants.
- Author the customer service agent in Copilot Studio using natural-language prompts and the visual canvas.
- Ingest SharePoint, OneDrive, and web sources; validate answers with the built-in simulator.
- Add conversation paths and escalation topics; practice handling edge cases.
- Publish the agent to Teams for testing and highlight deployment checklist.
- Q&A (5 minutes) and break (10 minutes).

## Segment 3 - HR Onboarding Workflow Build (60 Minutes)
- Translate onboarding process into topics, triggers, and entities.
- Layer prebuilt actions for email, approvals, and data updates; demonstrate Power Automate integration.
- Manage variables and decision logic to personalize onboarding experiences.
- Exercise: attendees extend the workflow with an additional action relevant to their org.
- Review analytics dashboards to verify data capture for optimization.
- Q&A (5 minutes) and break (10 minutes).

## Segment 4 - Autonomous Document Processor Build (60 Minutes)
- Configure event-driven triggers for email, file, and data updates to launch the document processor.
- Demo end-to-end automation: ingest documents, apply processing steps, and log outcomes.
- Address governance, security, and sharing controls before deployment.
- Measure ROI: interpret sessions, satisfaction, and agent-assisted hours to quantify value.
- Final exercise: identify three automation targets and draft follow-up actions.
- Wrap-up (5 minutes) with next steps and resource reminders.
