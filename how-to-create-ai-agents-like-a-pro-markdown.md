# How to Create AI Agents Like a Pro

**Publisher:** [Pearson](https://www.oreilly.com/publisher/pearson/)
**Level:** Intermediate
**Focus:** Use Microsoft Copilot Studio to build AI agents that automate business workflows without code.

## Overview
Microsoft Copilot Studio streamlines the creation of process automation agents. Across four hours, participants will craft three production-ready assistants for customer service, HR onboarding, and document processing while learning to monitor performance and quantify ROI.

## Key Outcomes
- Build agents with natural-language prompts and Copilot Studio’s visual tools.
- Connect agents to SharePoint, OneDrive, and web knowledge sources.
- Configure autonomous triggers for emails, files, and data events.
- Publish to Teams, SharePoint, and Microsoft 365 Copilot.
- Interpret analytics to optimize agent impact.

## Audience Fit
- Business analysts, project managers, and power users seeking no-code automation.
- Process owners tasked with improving operations without IT delays.
- Leaders managing customer service, HR, or operations workflows.

## Schedule

### Segment 1 — Fundamentals (50 minutes)
- Microsoft’s agent landscape
- Copilot Studio interface tour
- Demo: build your first agent
- Add knowledge sources and refine responses
- Exercise: FAQ agent
- Q&A (5 minutes) · Break (10 minutes)

### Segment 2 — Topics and Actions (50 minutes)
- Triggers, conversation paths, entities
- Prebuilt actions for email, approvals, and data updates
- Demo: HR onboarding workflow
- Power Automate integration
- Exercise: add an action
- Q&A (5 minutes) · Break (10 minutes)

### Segment 3 — Autonomous Agents (50 minutes)
- Preview features for event-driven automation
- Configure triggers for email, files, and data
- Demo: autonomous document processor
- Monitoring and governance considerations
- Exercise: configure a trigger
- Q&A (5 minutes) · Break (10 minutes)

### Segment 4 — Deployment and ROI (50 minutes)
- Publish to Teams and SharePoint
- Extend Microsoft 365 Copilot
- Analytics: sessions, satisfaction, topics
- Measure agent-assisted hours and cost savings
- Governance and roadmap planning
- Exercise: identify three automation targets
- Q&A (5 minutes) · Wrap-up (5 minutes)
