# How to Create AI Agents Like a Pro — Training Proposal

**Publisher:** [Pearson](https://www.oreilly.com/publisher/pearson/)
**Course Type:** Live online training (4 hours)
**Primary Tool:** Microsoft Copilot Studio

## Executive Summary
Microsoft Copilot Studio enables business users to automate customer service, HR, and document workflows without writing code. This four-hour session guides participants through building three production-ready agents, connecting them to enterprise data, and deploying across Microsoft 365 channels while tracking ROI through built-in analytics.

## Learning Outcomes
By the end of the engagement, participants will be able to:

- Build conversational agents with natural-language prompts and Copilot Studio’s visual authoring experience.
- Connect agents to organizational knowledge bases (SharePoint, OneDrive, websites, documents).
- Configure autonomous triggers that respond to email, file, or data events.
- Publish agents across Teams, SharePoint, and Microsoft 365 Copilot.
- Analyze agent performance metrics to drive continuous optimization.

## Target Audience
- Business analysts, project managers, and power users seeking no-code automation.
- Process owners accountable for operational efficiency without IT dependencies.
- Leaders overseeing customer support, HR, or operations where repetitive tasks dominate workloads.

## Detailed Agenda

### Segment 1 — Copilot Studio Fundamentals (50 minutes)
- Clarify Microsoft’s “agent” terminology and tooling landscape.
- Navigate Copilot Studio: home, create, analytics dashboards.
- Demo: authoring the first agent with natural-language descriptions.
- Add knowledge sources from SharePoint, OneDrive, websites, and documents.
- Test conversations in the simulator and refine responses.
- Mini-exercise: build an FAQ agent for a department.
- Q&A (5 minutes) and break (10 minutes).

### Segment 2 — Topics, Actions, and Power Automate (50 minutes)
- Model topics with triggers, conversation paths, and entities.
- Add prebuilt actions for email, approvals, and data updates.
- Demo: HR onboarding agent with approval workflows.
- Extend with Power Automate for advanced orchestration.
- Manage variables and decision logic without code.
- Mini-exercise: integrate a new action into participants’ agents.
- Q&A (5 minutes) and break (10 minutes).

### Segment 3 — Autonomous Agents and Event Triggers (50 minutes)
- Explore autonomous agent capabilities (preview feature).
- Configure triggers for email arrivals, file creation, and data changes.
- Demo: document processor agent that runs automatically.
- Monitor activity and interpret agent behavior dashboards.
- Address security and governance considerations.
- Mini-exercise: configure a trigger tailored to attendees’ scenarios.
- Q&A (5 minutes) and break (10 minutes).

### Segment 4 — Deployment, Analytics, and ROI (50 minutes)
- Publish agents to Teams and SharePoint.
- Extend Microsoft 365 Copilot with custom agents.
- Review analytics: sessions, satisfaction scores, topic performance.
- Quantify outcomes via agent-assisted hours and cost savings.
- Outline governance policies and sharing controls.
- Guided exercise: identify three candidate processes for automation.
- Q&A (5 minutes) plus final wrap-up (5 minutes).
